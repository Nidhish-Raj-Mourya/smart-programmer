# Smart Programmer

DSA visualizer platform — precomputed step playback for loops, arrays, trees, graphs, and more.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Live site

**https://nidhish-raj-mourya.github.io/smart-programmer/**

Repo: [github.com/Nidhish-Raj-Mourya/smart-programmer](https://github.com/Nidhish-Raj-Mourya/smart-programmer)

Deploys automatically on every push to `main` via GitHub Actions.

## Module 0 — Loop Foundations (implemented)

- **27 starter problems** across Levels 1–8 (foundation set; 109 total planned)
- **5 renderers:** CounterCanvas, DigitCanvas, GcdCanvas, SequenceCanvas, OutputBuilderCanvas
- **Movement types:** A, Q, R, S, T
